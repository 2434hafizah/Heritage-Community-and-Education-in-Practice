Why documentation matters
Engagement does not end when an activity finishes. Documentation preserves learning and experience so future initiatives can build on what has already been done.

Project Overview

This project documents an academic–community engagement related to heritage in Kuching. Rather than creating new activities, the project focuses on capturing learning, participation, and collaboration using existing materials and brief reflections. The documentation is designed to be ethical, simple, and practical.

Project Focus

The project focuses on:

  • Documenting engagement experience and learning moments
  • Preserving practice-based knowledge related to heritage education
  • Ensuring project outputs remain accessible beyond the programme period

Project Outputs

The documentation process produced the following outputs:

  • Flip Book – visual documentation of engagement activities and learning
  • Manual Book – short guide capturing key lessons and engagement principles
  • Website Repository – this website as a digital home for all outputs
  • Reflection – a brief written reflection on practice and learning

Ethical Approach

All materials are shared respectfully. Reflections are summarised, and no sensitive or identifying information is disclosed. Documentation prioritises learning and community dignity.
